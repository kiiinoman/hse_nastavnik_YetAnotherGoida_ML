# Задание 1. Аналитика добычи

Компания получает телеметрию со скважин:

* дебит;
* давление;
* температуру;
* энергопотребление;
* данные о простоях оборудования.

## Требуется построить аналитические витрины:

1. Общая добыча по дням;
2. KPI по скважинам:

   * средний дебит;
   * процент простоя;
3. Аналитика:

   * лучшие и худшие скважины;
   * влияние температуры и давления на добычу.

## Визуализации в Superset:

* Line Chart — добыча по времени;
* Bar Chart — топ скважин;
* Heatmap — давление vs дебит.

# Задание 2. ML-прогнозирование дебита

На основе таблицы `well_targets` требуется предсказать суточный дебит по параметрам:

* давление;
* температура;
* мощность;
* время работы насосов.

## Необходимо:

1. Подготовить датасет:

   * агрегировать telemetry по дням;
2. Обучить модель:

   * Linear Regression;
   * RandomForest;
3. Выполнить оценку:

   * MAE;
   * RMSE;
4. Построить прогноз.

## Визуализации в Superset:

* Actual vs Predicted;
* ошибка модели во времени.
